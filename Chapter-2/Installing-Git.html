安装 Git

Git 是一个分布式版本控制软件。

由于 Odoo 的源码获取和一系列其他工具的源码获取都需要用到 Git，所以我们先安装 Git。

Ubuntu 安装 Git

$ sudo apt-get update
$ sudo apt-get install git

Mac 安装 Git

$ brew install git

💡 Mac 安装 Git 使用的是 Homebrew —— macOS 缺失的软件包的管理器，安装方法详见Homebrew 官网。
